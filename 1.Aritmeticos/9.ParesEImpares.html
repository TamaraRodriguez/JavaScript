NUMEROS PARES E IMPARES

Realiza un algoritmo para un progama que lea 5 números. A continuación, nos mostarará la suma de todos los números pares y la suma de todos los números impares que introduzca que usuario.

OPERACIONES Y PASOS EN JAVASCRIPT

Nos iran apareciendo ventanas emergentes hasta llegar a completar los 5 números que nos pide el ejercicio.

var pos = 5;

var pares = 0;

var impares = 0;

for(i = 0; i < pos; i++){

var num1 = prompt("Introduce un número");

num1 = Number(num1);

if(num1 % 2 == 0){

pares = pares + num1;

}

else{

impares = impares + num1;

}

}
